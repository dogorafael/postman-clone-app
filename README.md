# PostmanClone
PostmanClone provides a user-friendly interface for making GET requests to APIs and viewing the responses in formatted JSON. Ideal for developers ans testers, this tool simplifies API interaction and debgging, helping you efficiently validate and troubleshoot your API calls.

## Technologies Used
* C#
* .NET 8
* WinForms
* HttpClient
